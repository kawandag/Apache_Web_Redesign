ABOUT US

Launched in 1995, the Apache HTTP server Project is an effort to develop and maintain an open-source HTTP server for modern operating systems incluing UNIX and Windows. The goal of this project is to provide a secure, efficient and extensible server that provides HTTP services in sync with the current HTTP standards. Our continous quest for improvement drives us to enhance our offerings to better serve the our clients around the world. Every year, the Apache Community Over Code unite for a four in-person days of sessions at one of our Denver, Europ and Asia to focus on Search, Big Data, Internet of Things, Community, Geospatial, Financial Tech, and many other topics.
